# GRUB PASTE - React App

GRUB paste is a web application to share a personal code inspired in Pastebin.

# 🚀 Features
- ⚙️ Support to C++, Python, Java and Plain text
- 🔑 Cryptography on client-side

# ⚡️ Getting started
1. Clone this repository
2. Installing
 ```sh
   yarn install
```
3. Running
 ```sh
   yarn start
```
